Android通过匹配Data启动Activity

2020-03-22
  • Android

总有一些蜜汁坑点.

调用方法

所谓Data即Uri.parse的参数scheme://host:port/path,

看起来很像HTTP链接对吧?

字段说明

key description
scheme 协议
host 主机名
port 端口号
path 这里包括前面的/, 之所以不删是为了与port区分

AndroidManifest.xml

虽然data里面有好几个属性, 但实际上除了scheme, 其他字段都可以省略(如果想被其他app随意调用的话).

但这样远远不够, 一调用就会马上出错,

还需要在intent-filter下添加action与category标签, 其name属性值可以任意.
