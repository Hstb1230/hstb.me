16/10/2补充: 结果到现在还是老样子

说明

  • 基于其他的聊天接口，只需一种调用格式即可正常使用

  • 总共有6个接口，分别是萌萌，青云，图灵，小黄鸡（SimSimi），茉莉，小i。
    补充: 8月份时新增 奇智和小冰 接口。

接口

提交

  1. 请求地址：http://chat.1sls.cn/v1/

  2. 请求方式：GET // Msg参数可用Post提交

  3. 必要请求参数：

  • type //接口类型，用数字0-8代替

  • msg //聊天内容，需utf8编码

  • id //用户ID
     

示例链接(返回内容需UTF-8解码):
http://chat.1sls.cn/v1/?type=1&id=123456&msg=233

说明

如果没有提交次要参数，则由接口自动生成，

返回文本为json格式，Status值为200即成功，
